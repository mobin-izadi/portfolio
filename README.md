## 🚀نصب و راه‌اندازی

 **کلون کردن مخزن:**

   ابتدا مخزن را کلون کنید و به دایرکتوری پروژه بروید:

   ```bash
   git clone https://github.com/mobin-izadi/portfolio.git
   ```

**نصب وابستگی‌ها:**
   

در داخل دایرکتوری پروژه، دستور زیر را برای نصب وابستگی‌های لازم اجرا کنید:

```bash
npm install
```

پروژه آماده استفاده است.
